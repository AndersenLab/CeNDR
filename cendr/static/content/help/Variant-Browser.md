# Variant Browser

<a name="standard-tracks"></a>
### Standard Tracks

The genome browser provides five tracks.

##### Genes 

Shows _C. elegans_ genes.

##### Transcripts

Shows transcripts of genes.

##### PhyloP

PhyloP scores measure evolutionary conservation. Positive scores suggest conserved sites. Negative scores suggestConservation across the following species:

* _Caenorhabditis elegans_
* _Caenorhabditis brenneri_
* _Caenorhabditis japonica_
* _Caenorhabditis remanei_
* _Caenorhabditis briggsae_
* _Strongyloides ratti_
* _Onchocerca volvulus_
* _Brugia malayi_



##### PhastCons

Conservation probability within the Caenorhabditis calculated by phastCons.
